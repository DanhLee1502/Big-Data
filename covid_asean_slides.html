🐳 namenode — bde2020/hadoop-namenode · Port 9870
Não của HDFS: lưu toàn bộ metadata (cây thư mục, vị trí block, quyền truy cập). Không lưu dữ liệu thực tế. Web UI tại localhost:9870 cho phép duyệt file system trực quan.
🗄️ datanode — bde2020/hadoop-datanode
Kho chứa thực sự: lưu block dữ liệu 128MB vật lý. Gửi heartbeat lên NameNode mỗi 3 giây. Nhân bản block để chịu lỗi. Giao tiếp nội bộ với namenode.
⚡ spark-jupyter — jupyter/pyspark-notebook · Port 8888
Môi trường PySpark: Python + Spark 3.5.0 + Java 17 cài sẵn. Volume mount ./work:/home/jovyan/work đồng bộ code VSCode↔container tự động — lưu file trong VSCode là chạy được ngay trong Spark.
📦 Parquet — Tại sao không dùng CSV?
Parquet: lưu theo cột, nén 60–80%, đọc nhanh 3–5×, schema tích hợp sẵn
CSV: lưu theo hàng, không nén, mọi tool đọc được — chỉ dùng để export kết quả cuối

Slide 04b · Kỹ Thuật
Window Functions — Kỹ Thuật Nâng Cao PySpark
LAG · RANK · ROW_NUMBER — không thể thay bằng GROUP BY thông thường
Khái niệm: Window Function tính toán cho từng hàng dựa trên "cửa sổ" hàng liên quan, nhưng GIỮ NGUYÊN tất cả hàng gốc — GROUP BY thì mất thông tin chi tiết (gom nhóm). Cửa sổ = PARTITION BY (nhóm) + ORDER BY (thứ tự).
LAG( ) — Nhìn về quá khứ
window = Window
  .partitionBy('location_key')
  .orderBy('year', 'month')

lag('tong_ca_moi', 1).over(window)
Lấy giá trị ca mắc của tháng TRƯỚC cùng quốc gia → tính MoM Growth Rate = (tháng này − tháng trước) / tháng trước × 100%
🇻🇳 VN 5/2021: 325 ca → 4.683 ca → MoM +1.341% → Delta xâm nhập!
🇻🇳 VN 3/2022: 1.270.879 → 6.093.034 → MoM +379% → Đỉnh Omicron
RANK( ) — Xếp hạng động
window = Window
  .partitionBy('year', 'month')
  .orderBy(desc('tong_ca_moi'))

rank().over(window)
Xếp hạng 11 nước theo ca mắc trong từng tháng riêng biệt. Tie → cùng rank. Cho phép theo dõi tâm dịch dịch chuyển theo thời gian.
🇮🇩 Indonesia #1: 11 tháng (2020–2021)
🇻🇳 Việt Nam #1: 6 tháng liên tiếp (11/2021–4/2022)
ROW_NUMBER( ) — Lấy đỉnh chính xác
window = Window
  .partitionBy('location_key')
  .orderBy(desc('tong_ca_moi'))

row_number().over(window)
.filter(col == 1) # Lấy 1 hàng
Đánh số duy nhất (không trùng) → filter == 1 để lấy đúng tháng cao nhất của mỗi nước. Khác RANK: kể cả tie cũng chỉ lấy 1 hàng.
